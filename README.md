# 🚀 EduNex – Interactive Learning & Mentorship Platform

EduNex is a full-stack **MERN-based e-learning platform** designed to connect students and instructors in a structured and interactive learning environment. It provides features like course management, mentorship sessions, progress tracking, certificates, and smart learning dashboards.

---

## 🌐 Live Demo
https://edunexplatform.vercel.app/

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- React Router
- Axios
- Context API
- Tailwind CSS

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Bcrypt.js

**Deployment:**
- Frontend: Vercel  
- Backend: Render  

---

## ✨ Features

### 👨‍🎓 Student
- Course browsing & enrollment
- Learning dashboard (progress, stats, recommendations)
- Consultation booking system
- Certificates after completion
- Course discussions & announcements

### 👨‍🏫 Instructor
- Create & manage courses
- Add lessons with lesson locking system
- Schedule consultations
- Instructor dashboard with analytics
- Manage enrolled students

### 🔐 Authentication
- Secure login/register system
- Role-based access (Student / Instructor / Admin)
- JWT authentication
- Password hashing with bcrypt
